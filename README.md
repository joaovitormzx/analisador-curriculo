# 📄 Analisador de Currículo vs Vaga

Ferramenta desenvolvida em Python que compara um currículo 
com a descrição de uma vaga e gera um score de compatibilidade 
com sugestões de melhoria.

## 🎯 O que o projeto faz

- Extrai palavras-chave da vaga e do currículo
- Calcula score de compatibilidade (%)
- Identifica palavras que faltam no currículo
- Sugere melhorias específicas
- Exporta o resultado em arquivo .txt

## 🛠 Tecnologias utilizadas

- Python 3
- Google Colab
- Bibliotecas: `re`, `collections`

## 🚀 Como usar

1. Abra o arquivo no Google Colab
2. Cole a descrição da vaga
3. Cole o texto do seu currículo
4. Execute as células em ordem
5. Veja o score e as sugestões geradas

- Manipulação de strings e expressões regulares
- Uso de Counter para análise de frequência
- Lógica de comparação de conjuntos
- Exportação de arquivos com Python
